# Spine 预览工具

前一阵子同事说 CCC 想看骨骼动画太麻烦了，能不能搞个预览工具。

好人有好报，这几天闲下来了就给他送上一份薄礼，请看 [SpinePreviewer ](https://github.com/DoooReyn/SpinePreviewer)

## 预览

![Spine 预览器](https://github.com/DoooReyn/SpinePreviewer/blob/main/previewer.gif)

## 使用

拖拽三个骨骼文件到窗口即可。

## 注意

- 目前只支持 json 格式的骨骼数据
- 目前只支持 spine 3.8 运行时
